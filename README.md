# :cloud: DarkSkyAPI

## Unofficial Dark Sky weather API in python
Dark Sky has recently stopped providing new API keys for their platform. This python module will allow querying Dark Sky using web scraping

##### Dependencies: 
* requests
* bs4
* geopy
